# ✨ Neon Cursor Effect Pro

A premium, highly interactive, and visually stunning neon cursor trail built entirely with Vanilla JavaScript and HTML5 Canvas. This zero-dependency project features physics-based particle tracking, dynamic color themes, magic sparkles, and a sleek glassmorphism control panel.

🔗 **Live Demo:** [Experience the Neon Cursor Pro!](https://pankajtiwari-art.github.io/Neon-Cursor-Effect-Pro/)

---

## 🌟 Key Features

* **Physics-Based Animation:** Utilizes linear interpolation (Lerp) for buttery-smooth pointer tracking and trailing effects.
* **🎨 5 Dynamic Color Themes:** * 🌌 **Velocity:** Cyan to Pink based on mouse speed.
  * 🌈 **Rainbow:** Continuous color cycling based on time.
  * 🔥 **Fire:** Red to Yellow heat mapping.
  * 🌇 **Sunset:** Beautiful Orange to Purple gradients.
  * 🌊 **Ocean:** Deep Blue to glowing Aqua.
* **✨ Magic Sparkles:** Emits glowing dust particles when moving the cursor quickly.
* **🪄 Blend Modes:** Toggle between "Lighter" (additive blending for extreme glow) and "Normal" rendering.
* **💤 Smart Sleep Mode:** When idle for 2 seconds, the cursor automatically animates in perfect mathematical patterns (Circle or Infinity).
* **🎛️ Pro Control Panel:** A fully responsive, slide-out glassmorphism settings menu to tweak trail length, thickness, glow intensity, and physics values in real-time.

---

## 🛠️ Tech Stack

* **HTML5 Canvas:** For high-performance 2D rendering.
* **CSS3:** Advanced Glassmorphism, CSS Variables, and smooth transition animations.
* **Vanilla JavaScript (ES6+):** Pure custom math, physics, and particle arrays—**Zero external libraries or dependencies!**
